## ⏱️ تتبع الدور في العيادة (Queue)

### كيف يعمل:

**1. عند حجز موعد:**
- النظام تلقائياً يضيف المريض للـ Queue
- حالة: `Waiting`

**2. عند وصول المريض للعيادة:**
- الـ Receptionist يعمل Check-in من لوحة التحكم
- حالة: `Checked In`

**3. عند استدعاء المريض:**
```php
// من لوحة التحكم أو عبر API
Queue::find($id)->call();
```
- حالة: `Called`
- 📱 يصل إشعار WhatsApp للمريض

**4. المريض يدخل للدكتور:**
- حالة: `In Consultation`

**5. بعد انتهاء الكشف:**
- حالة: `Completed`

### متابعة الدور عبر WhatsApp:

**المريض يبعت:**
```
دوري فين؟
```

**البوت يرد:**
```
⏰ دورك في العيادة:

📍 عيادة القلب
🔢 رقمك في الدور: 5
⏳ عدد المنتظرين قبلك: 3
⏱️ الوقت المتبقي التقريبي: 45 دقيقة

حالتك: في الانتظار ⏰
```
