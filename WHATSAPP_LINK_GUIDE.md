# 📱 رابط WhatsApp المباشر للمرضى

## 🎯 الهدف
المرضى يفتحوا رابط ويبدأوا المحادثة مع البوت على طول بدون ما يكتبوا حاجة!

---

## ✅ الخطوات البسيطة:

### 1️⃣ اعرف رقم WhatsApp بتاعك

**من Meta Developers:**
1. افتح: https://developers.facebook.com/apps
2. اختار تطبيقك
3. من القائمة: **WhatsApp > Getting Started**
4. هتلاقي **"Test number"** أو **"Phone number"**
5. انسخ الرقم (مثلاً: **+201234567890**)

---

### 2️⃣ حط الرقم في `.env`

افتح ملف `.env` وضيف السطر ده:

```env
META_PHONE_NUMBER=201234567890
```

**⚠️ ملاحظة:** اكتب الرقم بدون علامة `+` وبدون مسافات

---

### 3️⃣ مسح الـ Cache

```bash
php artisan config:clear
```

---

### 4️⃣ شغّل السيرفر

```bash
php artisan serve
```

---

## 🌐 الصفحة الجاهزة للمرضى

### افتح في المتصفح:

```
http://127.0.0.1:8000
```

أو:

```
http://127.0.0.1:8000/book
```

---

## 📱 الرابط المباشر

**صيغة الرابط:**

```
https://wa.me/[رقم_بدون_+]?text=مرحباً
```

**مثال:**

لو رقمك: **+201234567890**

الرابط يبقى:

```
https://wa.me/201234567890?text=مرحباً
```

---

## 🎨 الصفحة فيها إيه؟

عند فتح: http://127.0.0.1:8000

**هتلاقي:**

✅ **زر كبير أخضر:** "ابدأ المحادثة على WhatsApp"
- لما المريض يضغط عليه، WhatsApp يفتح مباشرة
- المحادثة تبدأ تلقائياً مع البوت

✅ **QR Code:**
- المريض يمسحه من الموبايل
- WhatsApp يفتح على طول

✅ **شرح الخطوات:**
- كيف يحجز المريض موعده
- المميزات اللي هيحصل عليها

✅ **تصميم جميل:**
- responsive (يشتغل على كل الأجهزة)
- عربي 100%
- سهل الاستخدام

## 🎬 السيناريو الكامل

### ما يحصل لما المريض يفتح الرابط:

**1. الرابط يفتح:**
```
https://wa.me/201234567890?text=مرحباً
```

**2. WhatsApp يفتح تلقائياً** (على الموبايل أو الويب)

**3. رسالة "مرحباً" موجودة جاهزة** (المريض بس يضغط إرسال)

**4. البوت يرد فوراً:**
```
أهلاً بك في الشفاء الذكي! 👋

كيف يمكنني مساعدتك؟

1️⃣ حجز موعد
2️⃣ تتبع موعدي
3️⃣ استشارة طبية
4️⃣ الاستعلام عن روشتة
```

**5. المريض يختار رقم** (مثلاً: 1)

**6. عملية الحجز تبدأ!** 🎉

## 🚀 نشر الموقع (Production)

### لما تنشر الموقع على سيرفر حقيقي:

**1. غيّر `APP_URL` في `.env`:**
```env
APP_URL=https://smart-medical.com
```

**2. الرابط يبقى:**
```
https://smart-medical.com/book
```

**3. شارك الرابط مع المرضى** 📢

## 💡 نصائح إضافية

### رسالة ترحيب مخصصة:

```
https://wa.me/201234567890?text=أريد%20حجز%20موعد%20في%20عيادة%20القلب
```

**الرموز:**
- `%20` = مسافة
- `%0A` = سطر جديد

### أمثلة:

**حجز موعد:**
```
https://wa.me/201234567890?text=أريد%20حجز%20موعد
```

**استشارة:**
```
https://wa.me/201234567890?text=عندي%20سؤال%20طبي
```

**تتبع موعد:**
```
https://wa.me/201234567890?text=دوري%20فين؟
```

## ❓ أسئلة شائعة

### س: الرابط مش شغال؟
**ج:** تأكد من:
- الرقم صحيح في `.env`
- Webhook مفعّل في Meta
- Token صالح

### س: الرابط يفتح WhatsApp بس البوت مش بيرد؟
**ج:** تأكد من:
- WhatsApp Webhook شغال
- السيرفر شغال
- الـ Token صحيح

### س: عايز أغير الرسالة الافتراضية؟
**ج:** غيّر `text=` في الرابط:
```
https://wa.me/201234567890?text=الرسالة_الجديدة
```
